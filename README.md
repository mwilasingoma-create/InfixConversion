# Infix Expression Converter in Java

## Description
This project converts an infix expression into:

- Postfix Expression
- Prefix Expression

## Concepts Used
- Stack Data Structure
- Operator Precedence
- String Reversal
- Expression Conversion

## Example

Input:
A+B*C

Output:
Postfix Expression: ABC*+
Prefix Expression: +A*BC

## How to Run

1. Compile:

javac InfixConversion.java

2. Run:

java InfixConversion
